Windows installation

  1. Download and Install Java Development Kit (JDK) from oracle. It needs to be known that PySpark only admits JKD versions 8 and 11. The installer must be chosen depending on the architecture (32 o 64 bits).
  2. Click on the installer, the installation will begin. Once finished, open the command prompt and execute the command java -version to verify the installation
  3. Download Spark from Spark downloads page. It is highly recommended to download the last version available. Extract the files in the desired location.
  4. After downloading Spark, Winutils will need to be installed and placed in the correct folder. Make sure to download the version corresponding to your hadoop version. For example. If you downloaded hadoop 3.0.0, download the winutils for 3.0.0
  5. Create a new folder named after "Winutils" and create inside a "bin" folder. Move "winutils.exe" to this folder.
  6. In the search bar, write "Edit the system environment variables" and then click on "Environment Variables".
  7. Click on "New". A new variable will need to be created named after JAVA_HOME and the value will need to be the directory where Java was installed (where the "bin" folder for Java is located).
  8. Click on "New" again. A new variable will need to be created named after HADOOP_HOME and the value will need to be the directory where Winutils was installed (where the "bin" folder for Winutils is located).
  9. Create a new variable named after SPARK_HOME and the value will need to be the directory where Spark was installed (where the "bin" folder for Winutils is located).
  10. Add to the PATH variable %JAVA_HOME%\bin %SPARK_HOME%\bin. To do this, click on "Edit".
